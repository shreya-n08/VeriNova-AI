# VeriNova-AI
AI-Powered Smart Ticket Authentication & Fraud Detection Platform
Event ticket fraud is a growing challenge, leading to financial losses, unauthorized access, and poor attendee experience. Traditional QR code validation alone cannot detect duplicated, tampered, or suspicious tickets.

VeriNova AI addresses this problem by combining secure ticket management, QR code verification, and AI-driven fraud detection into a unified platform. The system enables organizers to verify ticket authenticity in real time while providing actionable security insights through an analytics dashboard.
